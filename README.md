# Xtream

##Xtream : Mobile Streaming Test
[![License: GPL v2](https://img.shields.io/badge/License-GPL%20v2-blue.svg)](https://img.shields.io/badge/License-GPL%20v2-blue.svg)

안드로이드에서 음원 업로드 및 실시간 스트리밍 테스트 애플리케이션입니다.

###Features
- 음성 녹음 지원
- 기타 플레이어와 같이 녹음 가능
- 녹음 파일 업로드
- 녹음 파일 다운로드
- 모바일 스트리밍 지원
- (예정) 원 음원과 비교 후 점수 제공
	- 랭킹시스템 도입 가능


###Platform
- Android
- Web (지원예정)
- iOS (지원예정)
